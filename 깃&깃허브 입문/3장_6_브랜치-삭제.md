## 병합이 끝난 브랜치 삭제하기
> 브랜치 병합 후 사용하지 않는 브랜치는 깃에서 삭제할 수 있다.

### `git branch -d 브랜치명` : -D를 사용하면 병합하지 않은 브랜치도 강제 삭제
```c#
$ git branch
* master
  o2
$ git branch -d o2
Deleted branch o2 (was 2b761a2).
$ git branch
* master
```
- 삭제한 브랜치는 같은 이름으로 다시 브랜치를 만들면 예전에 작업했던 내용이 그대로 나타남
- 브랜치를 삭제하는 것은 깃의 흐름 속에서 감추는 것임
