## Keras
- input : 입력층
    - shape : 입력의 크기를 튜플로 지정

<br>

- Dense : 밀집층
    - 뉴런의 개수
    - activation= 활성화 함수 (sigmoid, softmax, relu ...)
   
<br>

- Sequential : 신경망 모델 생성
    - 신경망 모델에 추가할 층을 리스트로 전달
   
<br>

- compile() : 손실 함수와 측정 지표 등을 지정
    - optimizer= 신경망의 가중치와 절편을 학습하기 위한 알고리즘 (Adam ...)
    - loss= 이진분류(binary_crossentropy),  
            다중분류(categorical_crossentropy, sparse_categorical_crossentropy, ...)
    - metrics=[] 훈련과정에서 측정하고 싶은 지표를 리스트로 전달
   
<br>

- fit(입력 데이터, 타깃 데이터, ...) : 훈련
    - validation_data=(검증 데이터, 정답 데이터) : 에포크마다 검증 손실 계산
    - epochs= 반복할 에포크 횟수 지정
    - verbose= 2로 설정하면 진행 막대를 빼고 출력됩니다.
   
<br>

- evaluation(입력 데이터, 타깃 데이터) : 모델 성능 평가

<br>

```python
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense, Input, Flatten

model = Sequential([
    Input(shape=(28,28)),
    Flatten(),
    Dense(100, activation='relu', name='hidden'),
    Dropout(0.3),
    Dense(10, activation='softmax', name='output')
], name='fashion_mnist_model')

model.summary()

model.compile(optimizer='Adam', loss='sparse_categorical_crossentropy', metrics=['accuracy'])
history = model.fit(x_train_scaled, y_train, validation_data=(z_scaled, z_train), epochs=20, verbose=2)

model.evaluate(z_scaled, z_train)
```

<br>

- sparse_categorical_crossentropy : 타깃값이 정수인 다중 분류일 경우 loss 매개변수
   
<br>

- relu : 이미지 분류 모델의 은닉층에서 많이 사용하는 활성화 함수  
    시그모이드 함수는 층이 많을수록 함수의 양쪽 끝에서 변화가 작기 때문에 학습이 어려워짐

<br>

- Flatten() : 다차원 데이터를 1차원으로 펼쳐주는 층

<br>

- Dropout(비율) : 은닉층에 있는 뉴런의 출력을 랜덤하게 꺼서 과대적합을 막는 기법

<br>
