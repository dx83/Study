## Convolution
- 합성곱
- 밀집층과 비슷하게 입력과 가중치를 곱하고 절편을 더하는 선형 계산
- 밀집층과 달리 입력 전체가 아니라 일부만 사용하여 선형 계산 수행

### 필터
- 밀집층의 뉴런
- 커널 : 필터의 가중치와 절편
    - 자주 사용되는 커널의 크기는 (3,3) 또는 (5,5)
    - 커널의 깊이는 입력의 깊이와 동일

### 특성 맵
- 합성곱 층이나 출링 층의 출력 배열
- 필터 하나가 하나의 특성 맵을 생성

### 패딩
- 입력 배열의 주위를 가상의 원소로 채우는 것
- same padding : 입력과 특성 맵의 크기를 동일하게 만들기 위해 입력 주위에 0으로 패딩하는 것
- valid padding : 패딩없이 순수한 입력 배열에서만 합성곱을 하여 특성 맵을 만드는 경우, padding=디폴트 값으로 설정됨

### 스트라이드
- 합성곱 층에서 필터가 입력 위를 이동하는 크기
- 일반적으로 1 픽셀을 사용

### 풀링
- 가중치가 없고 특성 맵의 가로세로 크기를 줄이는 역할
- 풀링 크기 = 스트라이드, 패딩 없음
- 특성 맵의 크기를 줄여 연산량과 과적합을 감소시키고, 중요한 특성을 유지하면서 위치 변화(이미지 ex.고양이 귀 왼쪽->오른쪽)에 덜 민감하게 만들어 줌

<br>

- 최대 풀링 (max pooling) : 특성 맵에 있는 값 중 최댓값 선택
- 평균 풀링 (average pooling) : 특성 맵에 있는 값의 평균을 계산하여 나온 값

<br>

```python
from keras import layers
layers.Conv2D(10, kernel_size=(3, 3), activation='relu', padding='same')
layers.MaxPooling2D(2, strides=2, padding='valid')
```
