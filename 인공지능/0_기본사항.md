- 객체 = 모델클래스() : 사용할 모델 객체 생성
- 객체.fit(문제, 정답) : 주어진 데이터로 알고리즘 `훈련`, 훈련 결과는 객체 내부에 저장됨
- 객체.score(문제, 정답) : `평가`, 문제와 정답을 넣어 얼마나 맞는지 정확도 반환

- 하이퍼파라미터 : 모델이 학습할 수 없고 사람이 알려줘야 하는 파라미터

## 표준편차
- 데이터들이 평균에서 얼마나 멀리 떨어져서 분포해 있는지를 나타내는 통계적 지표
- 표준편차가 작다: 데이터들이 평균값 근처에 몰려 있습니다. (예: 전교생의 키가 대부분 비슷함)
- 표준편차가 크다: 데이터들이 평균에서 멀리 퍼져 있습니다. (예: 전교생의 키 차이가 매우 다양함)
