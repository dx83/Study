## 학습 모델
```python
# 학습 모델 임포트
import sklearn.~~~
```
- 객체 = 모델클래스() : 사용할 모델 객체 생성
- 객체.fit(문제, 정답) : 주어진 데이터로 알고리즘 `훈련`, 훈련 결과는 객체 내부에 저장됨
- 객체.score(문제, 정답) : `평가`, 문제와 정답을 넣어 얼마나 맞는지 정확도 반환
- 객체.predict(예측하려는 값) : 원하는 수치를 넣어 해당값의 `예측`값 확인
- 하이퍼파라미터 : 모델이 학습할 수 없고 사람이 알려줘야 하는 파라미터

## 데이터 전처리
### column_stack(tuple)
- 전달 받은 리스트를 일렬로 세운 다음 나란히 연결 (2차원 배열로 바꿔줌)
```python
import numpy as np
np.column_stack(([1,2,3],[4,5,6]))
```

array([[1,4],[2,5],[3,6]])
