## Decision Tree
- 결정 트리
- 예/아니오 에 대한 질문에 이어나가면서 정답을 찾아 학습하는 알고리즘

```python
from sklearn.tree import DecisionTreeClassifier
dt = DecisionTreeClassifier(max_depth=3, random_state=42)
dt.fit(x_train_scaled, y_train)
dt.score(x_train_scaled, y_train), dt.score(x_test_scaled, y_test)
```

<br>

- 트리 시각화
```python
import matplotlib.pyplot as plt
from sklearn.tree import plot_tree
plt.figure(figsize=(10,7))
plot_tree(dt)
plt.show()
```
```python
plot_tree(dt, max_depth=1, filled=True, feature_names=['alcohol', 'sugar', 'pH'])
```
- max_depth=: 트리 단계 제한
- filled=True: 노드의 분류 결과에 따라 색상을 채워 직관적으로 보여줍니다.
- feature_names=[...]: 특성(변수) 이름을 영어/기호 대신 지정한 텍스트로 변경.

<br>

- 지니 불순도 (Gini Impurity) : "현재 노드의 데이터가 얼마나 섞여 있는가(순수하지 못한가)"를 측정하는 지표
    - 지니 불순도는 0에서 0.5 사이의 값을 가집니다. (이진 분류 기준)
        - 0 : 한 종류만 있는 상태
        - 0.5 : 반반씩 있는 상태, 가장 혼란스러운 상태
