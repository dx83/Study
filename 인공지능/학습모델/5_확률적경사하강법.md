## Stochastic Gradient Descent
- 확률적 경사 하강법
- 훈련 세트에서 샘플 하나씩 꺼내 손실 함수의 경사를 따라 최적의 모델을 찾는 알고리즘
- 미니배치 경사 하강법 : 샘플을 하나씩 사용하지 않고 여러 개 사용

<br>

- 손실 함수 : 확률적 경사 하강법이 최적화할 대상
- 에포크 (epoch) : 전체 샘플을 모두 사용하는 한 번 반복
- 신경망 모델에서 사용

<br>

```python
from sklearn.linear_model import SGDClassifier
sc = SGDClassifier(loss='log_loss', max_iter=10, random_state=42)
sc.fit(x_train_scaled, y_train)
sc.score(x_train_scaled, y_train), sc.score(x_test_scaled, y_test)
```
- loss='log_loss' : 로지스틱 손실 함수 지정
- max_iter : 반복 횟수
- random_state : 랜덤 시드 지정

<br>

```python
sc.partial_fit(x_train_scaled, y_train) # 1 에포크씩 훈련
sc.score(x_train_scaled, y_train), sc.score(x_test_scaled, y_test)
```

<br>

- 훈련이 많아질수록 과대적합이 된다.
