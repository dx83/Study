# YOLO
- You Only Look Once
- 객체 탐지 인공지능 모델

### 장점
- 압도적인 실시간 속도: 지연없이 실시간 처리
- 높은 맥락 이해도: 배경을 물체로 착각하는 오류가 훨씬 적음
- 뛰어난 범용성: 다양한 영역에서 쉽게 적용

### 사용 사례
- 자율주행 및 모빌리티: 전방의 차량, 보행자, 신호등, 표지판 실시간 인식
- 보안 및 관제: CCTV를 통한 침입자 감지, 화재/연기 탐지, 불법 주정차 단속
- 스마트 팩토리: 컨베이어 벨트를 지나가는 제품의 불량(스크래치, 파손) 검수
- 리테일: 무인 상점에서 고객이 집어 드는 상품 인식 및 결제 자동화
