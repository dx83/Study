## 라쏘 회귀 (lasso)
- 계수의 절대값을 기준으로 규제를 적용

<br>

```python
from sklearn.linear_model import Lasso
lasso = Lasso()     # 매개변수 alpha
lasso.fit(x_train_scaled, y_train)
lasso.score(x_train_scaled, y_train), lasso.score(x_test_scaled, y_test)
```
- lasso = Lasso(alpha=수치) : 하이퍼파라미터, 규제의 강도

### 적절한 alpha 값 찾기
```python
import matplotlib.pyplot as plt
train_score = []
test_score = []

alpha_list = [0.001, 0.01, 0.1, 1, 10, 100] # 후보군
for alpha in alpha_list:
    lasso = Lasso(alpha=alpha)
    lasso.fit(x_train_scaled, y_train)
    train_score.append(lasso.score(x_train_scaled, y_train))
    test_score.append(lasso.score(x_test_scaled, y_test))

plt.plot(alpha_list, train_score)
plt.plot(alpha_list, test_score)
plt.xscale('log')
plt.xlabel('alpha')
plt.ylabel('R^2')
plt.show()
```
- 두 그래프가 가장 가까울 때가 적절한 alpha값
