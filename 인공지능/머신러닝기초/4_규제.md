## 규제 (reqularization)
- 모델이 훈련 세트를 너무 과도하게 학습하지 못하도록 훼방하는 것

<br>

```python
from sklearn.preprocessing import StandardScaler
ss = StandardScaler()
ss.fit(x_train_poly)                        # 훈련 세트의 평균과 분산 계산
x_train_scaled = ss.transform(x_train_poly) # 훈련 데이터의 값들을 표준화
x_test_scaled = ss.transform(x_test_poly)   # 훈련 세트의 평균과 분산을 가져와서 테스트 세트도 똑같이 변환
```
- x_train_scaled : 모든 특성의 평균을 0, 분산을 1로 맞춘 새로운 데이터셋인 x_train_scaled를 만들어 반환
- x_test_scaled : 테스트 데이터의 평균을 새로 구하는 것이 아님
