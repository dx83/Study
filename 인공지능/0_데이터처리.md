## 데이터 탐색

```python

```

## 데이터 전처리
### column_stack(tuple)
- 전달 받은 리스트를 일렬로 세운 다음 나란히 연결 (2차원 배열로 바꿔줌)
```python
import numpy as np
np.column_stack(([1,2,3],[4,5,6]))
```
```
array([[1,4],[2,5],[3,6]])
```
