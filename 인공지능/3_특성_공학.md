## 특성 공학 (Feature Engineering)
- 주어진 특성을 조합하여 새로운 특성을 만드는 일련의 작업 과정

### PolynomialFeatures
- 선형 모델이 비선형적인 관계(곡선)도 학습할 수 있도록 특성을 부풀리는 역할
- include_bias=False: 기본값인 선형 회귀의 절편(상수항 1)을 생성하지 않음, 제곱과 교차항만 남음
- fit() : 특성 조합
- transform() : 실제로 데이터를 변환

```python
from sklearn.preprocessing import PolynomialFeatures
poly = PolynomialFeatures(include_bias=False)   # 절편 항 무시
poly.fit(x_train)
x_train_poly = poly.transform(x_train)
x_test_poly = poly.transform(x_test)
x_train_poly.shape, x_test_poly.shape
```
