# py(파이썬)으로 짠 윈도우 스크립트가 필요하다.

### 기본 목표
1. Efficientnet-b0로 학습시킬 알약이미지의 전처리
2. 각 알약 이미지에서 알약을 크롭해서 폴더를 만들어 그 안에 넣는다.

### 목표 상세
1. 크롭 방법
```
원본 이미지 27,000장 이상
            ↓
OpenCV Contour 검출
            ↓
알약 2개 검출 (3개이상도 있음)
            ↓
개별 Crop
            ↓
10~20% Padding
            ↓
PNG 저장
```

2. 원본 이미지 파일
- 파일
    - 예시) 200101951_한신타미놀캅셀(아세트아미노펜)_1pw5pyj7qmnxr01.jpg
    - 고유번호_알약이름_의 구조로 되어 있다.

- 폴더
    - images 폴더내 각 하위 폴더안에 알약아미지가 있다.
    - 폴더명 '01' ~ '28' 로 28개의 하위 폴더로 구성
    - 각 하위 폴더 안에 알약 이미지가 1000개씩 있다. (단, 마지막 폴더는 387개)

3. 디렉토리 구조화 (Directory Structuring)
- 크롭한 이미지는 원본과 다른 곳에 보관해야 할 것이다.
- images 폴더의 상위 폴더 위치에 images_crop 폴더를 만든다.
- images_crop 폴더내에 원본 이미지의 [고유번호_알약이름] 으로 폴더를 만든다.
- 해당 알약의 크롭한 이미지들을 [고유번호_알약이름_1.png], [고유번호_알약이름_2.png], ... 으로 해당 폴더에 저장한다.

4. py 파일 실행
- images 폴더의 상위 폴더 위치에서 cmd로 py 파일을 실행 할 것

### 최종
- 내가 필요한 최종 대답을 1장의 python 스크립트로 보여주면 된다.
