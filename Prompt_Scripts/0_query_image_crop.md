# 기본 목표
Efficientnet-b0로 학습시킬 원본 알약이미지의 가공이 필요하다.

## 목표 상세
0. 준비된 알약이미지 : 약 27000장

1. 각 알약이미지의 구성
- 알약 1개의 앞 뒤를 찎은 이미지
- 규격 선과 수치
- 식품의약품안전처 한글과 영어명 그리고 마크
- 배경 : 하늘색, 연한 하늘색, 분홍색 연분홍색 등

- 이미지 형태
----------------
- [알약] [알약] -
----------------

또는

----------------
-   [알약]
       [알약]  -
----------------

- 문제점
    - 규격선과 수치, 배경, 얄약 배치는 약간씩 다른게 있음
    - 이미지가 많아서 위의 규칙만 있는지는 사람이 다 확인 할 수 없음

2. 크롭할 대상 : 알약의 앞 뒤를 찍은 각 이미지

- 알약을 크롭이 아니라 단순하게 세로 혹은 가로로 자를 수도 있지만 위치가 애매해서 사선으로 잘라야 하는 경우도 있다.
- 이미지가 너무 많아서 사람이 그걸 분류해서 정리하기 힘들다.
- 그래서 한번에 자동으로 크롭을 해야 할 것으로 생각된다.

3. 크롭 방법 (가장 중요)
- 챗GPT가 제시하는 방법이 필요하다.
- 여러 의견을 찾아줘

### 추가로 고려 사항이 있는데, 내가 지금 1.모양 검출 2.색상 검출 3.알약문자/마크 검출을 따로 할 계획이다. 그래서 크롭된 알약이미지가 선명하게 남아 있었으면 한다.
