# 모델 B: 색상 분류

### 색상
- 하양, 분홍, 노랑, 주황, 갈색 등
- 각 색깔 진한 계열
- 각 색깔 투명 계열
- Two-tone Capsule (하양+초록, 하양+청록 등)

### 방법 1. OpenCV + HSV + LightGBM
```
이미지 -> 알약 영역 추출 -> HSV 변환 -> 평균 H, S, V -> LightGBM -> 색상 분류
```

장점
- 학습 속도 빠름
- 데이터 적어도 가능

단점
- 조명 영향 받음
- 진한/투명 구분 어려움
- 복잡한 투톤 조합에 한계
- 특징 설계 필요

### 방법 2. EfficientNetNet-B0 Multi-Output
`Multi-Output` 하나의 모델이 여러 개의 결과를 동시에 예측

```
이미지 -> EfficientNetNet -> 앞색상/앞농도/뒤색상/뒤농도
```

장점
- 특징 설계 불필요
- 복잡한 색 처리 가능
- 향후 품목 분류 모델로 확장 가능

단점
- 학습 시간 김
- 결과 해석 어려움
- 데이터 불균형 처리 필요
