# 모델 C: 글씨 및 마크 인식 (OCR 계열)

### 문자 / 마크

## 방법 1. PaddleOCR + OpenCV 전처리

```
알약 영역 크롭 -> 이미지 전처리(그레이스케일,대비 향상,샤프닝) -> PaddleOCR 수행 -> 텍스트 추출
```

장점
- 구현 쉬움
- 무료
- 한국어 지원
- 숫자/영문 인식 우수
- 학습 없이 바로 사용 가능
- 빠른 프로토타입 제작 가능

단점
- 작은 글씨에 약함
- 흐릿한 각인에 약함
- 음각/양각 인식 어려움
- 로고(마크) 인식 성능 낮음
- 글씨가 없는 경우 처리 어려움

## 방법 2. YOLO + OCR

```
YOLO로 각인 영역 검출 -> 검출 영역 크롭 -> PaddleOCR 또는 CRNN 수행 -> 문자 추출
```

장점
- OCR보다 높은 정확도
- 작은 각인에 강함
- 배경 영향 감소
- 알약 위치 변화에 강함
- 실제 서비스 구조와 유사

단점
- 라벨링 필요
- 구현 복잡
- 학습 시간 필요
