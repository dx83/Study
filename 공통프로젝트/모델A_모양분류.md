## 방법 1. OpenCV + Random Forest + 확률 임계값
`특징 설계` OpenCV
- 가로세로비
- 원형도
- 면적
- 둘레
- 꼭짓점 수

`Random Forest` 모양 분류

`기타 분류 문제` 최대 확률 < 0.7 => 기타

장점
- 구현 쉬움
- 데이터 적어도 됨
- 해석 가능

단점
- 모양 차이가 미묘한 클래스는 어려움
