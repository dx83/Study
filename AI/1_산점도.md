## 산점도
- 두 변수 사이의 관계를 알아보기 위해, 평면 좌표 위에 데이터를 점으로 흩뿌려 나타낸 그래프
- 점들이 구름처럼 사방으로 흩어져 있다면, 두 데이터의 연관성이 매우 약하다는 뜻

<br>

```python
import numpy as np
import matplotlib.pyplot as plt
plt.rc('font', family='Gulim')

plt.scatter(a, b, marker='^', c='r') # 삼각형, 빨강
plt.scatter(x_train, y_train, c='g') # 기본 원형, 초록
# 마름모, 파랑
plt.scatter(x_train[indexes], y_train[indexes], marker='D', c='b')

# x축 y축, 선그리기
point = np.arrange(15, 50)
plt.plot(point, 1.01*point**2 - 21.6*point + 116.05, c='black')

plt.xlabel('x축 이름')
plt.ylabel('y축 이름')
plt.show()              # 산점도 그리기
```
<br>

```python
plt.xlim((a,b))         # x축 범위를 a,b로 보이게 해줌
plt.ylim((c,d))         # y축 범위를 c,d로 보이게 해줌
```
