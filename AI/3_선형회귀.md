## Linear Regression
- 선형회귀

```python
from sklearn.linear_model import LinearRegression
lr = LinearRegression()

# 선형 회귀 모델을 훈련
lr.fit(x_train, y_train)

# 예측
lr.predict([[50]])
```
<br>

- lr.coef_ : 기울기, 계수 또는 가중치
- lr.intercept_ : 절편, 입력값이 0일 때 기본 결과값
```python
# a에서 b까지 1차 방정식 그래프를 그립니다.
plt.plot([a,b], [a*lr.coef_+lr.intercept_, b*lr.coef_+lr.intercept_])

# c에서 d까지 2차 방정식 그래프를 그립니다. 정답 = lr.coef_[0] x 문제^2 + lr.coef_[1] x 문제 + lr.intercept_
point = np.arrange(c, d)
plt.plot(point, 1.01*point**2 - 21.6*point + 116.05)
```
